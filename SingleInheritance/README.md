# Single Inheritance in C++
## Quick Revision Notes (GeeksforGeeks Style)

### Definition
In Single Inheritance, a class is allowed to inherit from only one class. i.e., one subclass is inherited by one base class only.

### Key Points
- The simplest form of inheritance.
- Prevents complex relationships and is easy to understand.

### Example Syntax
```cpp
class Base { };
class Derived : public Base { };
```
