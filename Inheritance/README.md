# Inheritance in C++
## Quick Revision Notes (GeeksforGeeks Style)

### Definition
Inheritance is the capability of one class to acquire properties and characteristics from another class. It is a fundamental concept of Object-Oriented Programming (OOP) in C++.

### Key Points
- The class that inherits properties is called the **Derived Class** or **Child Class**.
- The class whose properties are inherited is called the **Base Class** or **Parent Class**.
- It supports the concept of **Reusability**.
- Types: Single, Multiple, Multilevel, Hierarchical, Hybrid.

### Example Syntax
```cpp
class Base { public: int x; };
class Derived : public Base { public: int y; }; // Inherits x
```
